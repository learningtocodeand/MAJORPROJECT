A full-stack web application inspired by Airbnb, built using the MERN stack. The platform allows users to list rental properties, explore available stays, leave reviews, and view property locations on an interactive map.

## Features

- User authentication and authorization
- Create, view, update, and delete property listings
- Only authorized users can edit or delete their own listings
- Users can add reviews and comments to properties
- Interactive map integration using Mapbox API to display property locations
- Secure RESTful APIs with proper validation and error handling

## Authentication & Authorization

- User registration and login implemented
- Protected routes to restrict unauthorized access
- Ownership-based authorization to ensure users can modify only their own listings

## Tech Stack

- Frontend: React.js, HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: Passport.js / JWT
- Maps: Mapbox API
